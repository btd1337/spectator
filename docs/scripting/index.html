# Scripting

Spectator gives the user the ability to augment their requests with scripting. The scripting language used in Spectator is JavaScript. For execution of the scripts the emedabble JavaScript engine Duktape is used. Most of the provided API maps to native code.

The script is executed at a certain point, right before the execution of the Spectator request. In future there shall be a callback for intercepting the Spectator response object.
