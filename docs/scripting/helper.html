# Helper

# UrlEncoded(data)

data is of type object, i.e. a flat object.

Converts the object to a an UrlEncoded object. This object is only for the internals of Spectator and not usable in the JavaScript context

Example:

UrlEncoded({
    "url": "encoded",
    "some": "value"
})

# FormData(data)

data is of type object, i.e. a flat object.

Converts the object to a an FormData object. This object is only for the internals of Spectator and not usable in the JavaScript context

Example:

UrlEncoded({
    "form": "data",
    "funny": "stuff"
})
