# Usage

# Create a request

To create a new request either click on the star symbol on the left in the headerbar or the button in the welcome view

You can give the request an arbitray name, which doesn't have to unique. You can also select the method of your request. You can change both attributes later.

# Configure request

Now you can configure your request. At the top is a URI entry. To send a request the content has to be a valid http or https URL.

# Headers

Under the tab header you can additionally add custom headers like User-Agent or Authorization.

# Parameters

Adding parameters under the parameter tab adds a query to the URL.

# Body

The Body tab will only be active if the method POST, PUT or PATCH is selected.

You have the option to add Form-Data, UrlEncoded or text bodies.

Selecting raw from the dropdown gives you the choice of sending plain text, JSON, XML or HTML. The Content-Type of your request will be set accordingly.

# Script

You can also augment your request with additional functionality via JavaScript. There is more information in the scripting section.
